B-SMART

Bubble System for Monitoring and Rubbish Trapping

Inovasi sistem IoT yang dirancang untuk memantau dan menangkap sampah plastik di sungai secara otomatis menggunakan teknologi bubble barrier dan sensor load cell IoT yang terhubung langsung ke dashboard web real-time.

Desa Pejagan, Bangkalan, Jawa Timur
Mulai Beroperasi: 09 April 2026
Kompetisi JRC 13 – 2026

Monitoring Real-Time 24/7

Data berat sampah dan deteksi visual dikirimkan secara berkala melalui ESP32 ke server cloud Firebase tanpa downtime.

Sensor Load Cell IoT

Sensor berat Load Cell + HX711 mengukur berat sampah terkumpul secara akurat dan mengirimkan data real-time melalui ESP32 ke Firebase cloud.

Bubble Barrier Ramah Lingkungan

Teknologi tirai gelembung udara dari dasar sungai mengarahkan sampah tanpa membahayakan ekosistem air dan biota sungai.

Notifikasi Otomatis

Sistem mengirimkan notifikasi otomatis ke admin ketika kapasitas penampungan penuh atau terjadi anomali pada sensor.
